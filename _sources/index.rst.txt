
Welcome to Haskell Optimization Handbook
==========================================

This book assumes you are using ghc ``8.10.x`` or above and using the Linux
operating system. If you are new please start with the :doc:`Intended Audience
</src/Preliminaries/book_for>` and :doc:`How to use this book
</src/Preliminaries/how_to_use>` chapters. We wish you the best on your
optimization journey!
